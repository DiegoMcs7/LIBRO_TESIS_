## Como usar este material

1. Estudiar capitulo por capitulo, respondiendo las preguntas de cada seccion
2. Las preguntas estan ordenadas por categoria tematica dentro de cada archivo
3. Las preguntas globales (carpeta 08) cruzan multiples capitulos y son las mas probables en una defensa real
4. Prestar especial atencion a las categorias "preguntas desafiantes / devil's advocate"
5. Practicar respuestas orales con limite de tiempo (2-3 minutos por pregunta)
